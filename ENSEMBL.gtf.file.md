#### To overlap ClinVar missense variants genomic coordinates with coordinates in exon annotation file, gtf file below should be used
The Homo_sapiens.GRCh37.87.gtf.gz file provides detailed, tab-delimited genomic annotation data for the human GRCh37 assembly, including coordinates, structures, and metadata for genes, transcripts, and exons. 
https://ftp.ensembl.org/pub/grch37/release-98/gtf/homo_sapiens/Homo_sapiens.GRCh37.87.gtf.gz
