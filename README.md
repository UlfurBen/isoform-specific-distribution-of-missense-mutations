# The Isoform Specific Distribution of Missense Mutations

This Github repository contains code that was used to calculate the distribution of missense mutations in isoform specific areas (i.e. exons that belong to no more than 1 isoform) in around 300 developmentally and hopefully clinically relevant epigenetic machinery genes (i.e. genes that encode proteins responsible for regulating gene expression without altering underlying DNA sequence).
The motivation for this project was to find specific exons that belong to no more than 1 isoform and contain an unusually high density of pathogenic missense mutations.
